# 🤖 小小程序员逻辑思维训练器

> 翠苑第一小学教育集团信息素养提升实践活动 - 创意编程作品

## 📋 作品简介

这是一个专为小学三年级学生设计的编程思维训练工具。通过图形化的积木指令，学生可以控制机器人在迷宫中移动，学习编程的基本概念，同时实时看到对应的Python代码。

## ✨ 核心特色

### 1. 计算思维培养
- **顺序执行**：理解程序按顺序运行的概念
- **循环结构**：学习用循环简化重复代码
- **算法思维**：通过迷宫寻路培养逻辑推理能力

### 2. 代码可视化
- 拖拽积木的同时，实时生成对应的Python代码
- 让学生看到图形化指令与真实编程语言的对应关系
- 为未来学习Python打下基础

### 3. 渐进式学习
- 6个精心设计的关卡，从简单到复杂
- 涵盖顺序、转向、循环等多种编程概念
- 障碍物设计培养问题解决能力

### 4. 成就系统
- 完成关卡获得成就徽章
- 激励学生持续学习和探索

## 📱 使用说明

1. **选择关卡**：从关卡列表中选择一个已解锁的关卡
2. **添加指令**：点击积木区的指令按钮添加到程序区
3. **使用循环**：
   - 点击"重复"积木添加到程序区
   - 点击重复积木进入编辑模式（会出现黄色边框）
   - 此时点击"前进/左转/右转"指令，会自动添加到重复积木中
   - 点击⚙️图标可设置重复次数（1-10次）
   - 点击"完成"退出编辑模式
4. **查看代码**：在Python代码区查看对应的代码
5. **运行程序**：点击"运行程序"按钮执行代码
6. **观察结果**：观看机器人按照指令移动
7. **通关成功**：到达终点即可通关，获得成就！
